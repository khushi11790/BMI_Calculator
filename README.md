# 🧮 BMI Calculator

## 🚀 Overview

This is a simple BMI (Body Mass Index) Calculator built using React. It allows users to input their height and weight to calculate BMI and determine their health category.

## ✨ Features

* 📏 Input height and weight
* ⚡ Instant BMI calculation
* 📊 Displays health category (Underweight, Normal, Overweight, Obese)
* 🎨 Simple and responsive UI


## 🛠️ Tech Stack

* React.js
* JavaScript
* HTML
* CSS

## ▶️ Run Locally

npm install
npm start


## 📊 BMI Formula

BMI is calculated using:
BMI = weight (kg) / height² (m²)
